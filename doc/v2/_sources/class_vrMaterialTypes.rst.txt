    .. _vredvrMaterialTypes.MergeOptions:
    .. py:class:: MergeOptions

        Defines options for merging materials.


            Default, compare name and and skip Switch and X-Rite materials.

            .. _vredvrMaterialTypes.MergeOptions.Default:
            .. py:data:: Default


            Do not compare the name of the material, i.e., materials with different names can be considered equal.

            .. _vredvrMaterialTypes.MergeOptions.IgnoreName:
            .. py:data:: IgnoreName


            Also merge Switch and X-Rite materials.
