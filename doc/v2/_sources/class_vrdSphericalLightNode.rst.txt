---------------------
vrdSphericalLightNode
---------------------


.. _vredvrdSphericalLightNode:
.. py:class:: vrdSphericalLightNode

(Inherits :py:class:`vrdAreaLightNode`)

This class gives access to the properties of a spherical light source.

The spherical light is an area light that emits light from a sphere. It is more realistic than point light.
