vrdMultiMarker

.. _vredvrdMultiMarker:
.. py:class:: vrdMultiMarker

(Inherits :py:class:`vrdMarker`)

Multi Marker.

This is a marker that is combined of multiple markers of the same type. The confidence and the pose of the markers is averaged for the multi marker.
