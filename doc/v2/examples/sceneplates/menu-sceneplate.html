Sceneplate with HTML menu¶

You can interact with the scene from within an HTML page by calling Python functions of the VRED Python API, as well as custom Python functions defined in the script that comes with the vpb file (Edit > Script Editor).

Use JavaScript to execute Python code in VRED:

JavaScript¶
// just calls some Python code
vred.executePython("setWireframe(SWITCH_TOGGLE)");

// example how to call a Python command with a return value
vred.executePythonCommand("getFov()", function(value)
  { alert(value); } // do something with the value returned by getFov()
);

vred is a special object that is available in JavaScript when running the web page within a VRED web engine. The Python code passed to executePython and executePythonCommand is executed by VRED’s Python interpreter, similar to running it in the Terminal or Script Editor.

Example¶

The example file menu-sceneplate.vpb shows an HTML menu on a Frontplate (Sceneplate Editor). A click on the buttons triggers an action in the VRED scene, like switching the material and playing an animation.

The Python code in the vpb (Edit > Script Editor) defines functions that are called when clicking on the Top, Left, Bottom, Right buttons in the menu:
